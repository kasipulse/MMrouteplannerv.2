## Features

## Setup

1. Clone repo
2. Create Firebase project, enable Auth & Firestore
3. Replace `YOUR_API_KEY` and `YOUR_GOOGLE_MAPS_KEY` in `index.html` & `main.js`
4. Deploy web folder via Firebase Hosting
5. Optional: Android Studio WebView or native Maps SDK
